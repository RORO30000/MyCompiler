# MyCompiler

Compilador simple desarrollado en C++.

---

# Cómo usar el proyecto

## 1. Compilar

Abre una terminal dentro de la carpeta del proyecto y escribe:

make

Esto generará el ejecutable llamado:

compilador

---

## 2. Ejecutar

Después de compilar escribe:

make run

---

## 3. Compilar y ejecutar en un solo comando

make go

---

## 4. Limpiar archivos compilados

Para borrar los archivos .o y el ejecutable:

make clean

---

# Estructura del proyecto

main.cpp      -> Programa principal
lexer.cpp     -> Análisis léxico
parser.cpp    -> Análisis sintáctico
Makefile      -> Automatiza compilación

---

# Tecnologías utilizadas

- C++
- STL
- Makefile
